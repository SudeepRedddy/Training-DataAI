# Netflix Data Analysis Project

This project performs a comprehensive analysis of the Netflix dataset using NumPy, Pandas, and Matplotlib, covering 38 specific assignments including data cleaning, manipulation, visualization, and reporting.

## Prerequisites

1.  **Python 3.x** installed.
2.  **Netflix Dataset**: Download `netflix_titles.csv` from Kaggle and place it in this directory.

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Usage

Run the main analysis script:

```bash
python main.py
```

## Outputs
*   **Plots**: Generated charts are saved in the `plots/` directory.
*   **Report**: A markdown report is generated at `output/analysis_report.md`.
*   **Data**: The processed dataset is saved to `output/processed_netflix_data.csv`.
